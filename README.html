System Architecture

How Kavach is built

A fully integrated pipeline — from real-time data ingestion, through the Zone Intelligence AI engine, to instant UPI payouts.

AI / ML Integration

Intelligence at every layer

Four distinct AI/ML modules, each solving a concrete problem in the Kavach pipeline.

01

Dynamic Premium Calculator

Gradient Boosted Regressor trained on synthetic historical disruption data for 10 Indian cities. Adjusts weekly premium per zone based on flood frequency, season, and rider tenure.

XGBoost scikit-learn Zone features
02

Disruption Classifier

Zero-shot text classifier on live news/govt alerts. Identifies disruption type, affected zone, and confidence score. Keyword-rule fallback for demo reliability.

HuggingFace BART-large-MNLI NLP
03

Fraud Guard — Anomaly Detection

GPS trace analysis to detect spoofing. Cross-checks rider location against disruption zone boundary. Anomaly scorer flags new-policy claims in unusual micro-events for review.

Isolation Forest GPS clustering Rule engine
04

Predictive Risk Forecasting

Time-series model forecasting next-week disruption probability per zone. Powers insurer dashboard with reserve recommendations: "Zone 7 has 68% chance of rain trigger next week."

Prophet / LSTM Time series Zone forecasting
